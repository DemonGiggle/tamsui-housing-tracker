下一步擴充建議

  1. 保留目前社區頁來源，持續抓社區級價格資訊與欄位。

  3. 原始資料層維持 source、source_url、raw_hash，後續補上 fetched_at、來源型別與比對信心欄位。
  4. 展示層只使用能解釋來源的資料，不再用 fallback 補滿畫面。
